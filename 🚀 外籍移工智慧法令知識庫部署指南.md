# 🚀 外籍移工智慧法令知識庫部署指南

這套系統專為 **Streamlit Community Cloud** 設計，您可以完全免費地將其部署為一個永久網頁。

## 🛠️ 快速部署步驟 (只需 3 分鐘)

### 第一步：準備 GitHub 倉庫
1. 登入您的 [GitHub](https://github.com/) 帳號。
2. 建立一個新的私有或公開倉庫（例如命名為 `migrant-kb`）。
3. 將以下三個檔案上傳到該倉庫的根目錄：
   - `app.py` (主程式)
   - `database.csv` (初始資料庫結構)
   - `requirements.txt` (環境設定檔)

### 第二步：部署至 Streamlit Cloud
1. 前往 [Streamlit Community Cloud](https://share.streamlit.io/)。
2. 點擊 **"Create app"**。
3. 選擇您剛才建立的 GitHub 倉庫、分支（通常是 `main`）以及主檔案 `app.py`。
4. 點擊 **"Deploy!"**。

### 第三步：開始使用
* 部署完成後，您會得到一個專屬網址（例如 `https://migrant-kb.streamlit.app`）。
* **手機存取：** 直接用手機瀏覽器開啟該網址，並將其「加入主畫面」，就像使用 App 一樣直覺！

---

## 💡 使用小撇步
* **資料儲存：** 在 Streamlit Cloud 上，如果您重新部署或重啟 App，暫存的 CSV 資料可能會重置。建議定期使用系統內的「下載 CSV」按鈕備份資料。
* **AI 解析：** 匯入 LINE 紀錄時，請務必填入您的 OpenAI 或 Gemini API Key。
